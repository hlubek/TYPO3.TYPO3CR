javax.naming.directory.DirContext 
getDirectory() 
Returns the underlying JNDI directory, if available. Otherwise returns 
null.

RepositoryException is thrown if an error occurs. 
java.security.Principal 
lookupPrincipal(String principal) 
Looks up a principal by the string representation used in the 
jcr:principal property of an nt:ace node. 
The following should be true: 
lookupPrincipal(session.getUserID()). 
getName().equals(session.getUserID()) 
Returns 
null if the principal is not found. 
A 
RepositoryException is thrown if an error occurs. 
PrincipalIterator 
searchPrincipal(String simpleFilter) 
Gets the principals matching a simple filter expression applied against 
the string representation of a principal (the value used in the 
jcr:principal property of an nt:ace). 
A star " 
*" in the filter expression matches zero or more characters. "\*" 
matches a literal " 
*". 
An implementation may limit the number of principals returned. 
If there are no matching principals, an empty iterator is returned. 
A 
RepositoryException is thrown if an error occurs.
