== Row ==

=== Value[] getValues() ===
Returns an array of all the values in the same order as the column 
names returned by QueryResult.getColumnNames(). 
Throws a RepositoryException if an error occurs. 

=== Value getValue(String propertyName) ===
Returns the value of the indicated  property in this Row. 
If propertyName is not among the column names of the query result 
table, an ItemNotFoundException is thrown. 
Throws a RepositoryException if another error occurs.
